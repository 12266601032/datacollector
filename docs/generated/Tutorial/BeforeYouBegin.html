Before You Begin

Before you start this tutorial, you'll need to do a few things:

  1. Download sample data.

    You can download sample data from the following location:

    https://www.streamsets.com/documentation/datacollector/1.1.0/tutorial/sample_data/nyc_taxi_data.csv

    You'll need a login to the password-protected section of the site - it just takes a moment to set up an account.

  2. Set up a directory local to the Data Collector, and save the sample data in the directory.

    Let's use the same directory for the source, destination, and error record files. Make sure the user who started the Data Collector has read and write permission for the directory.

  3. The Data Collector should be installed and running.
  4. To create and run the pipeline, you should have a Data Collector login with the admin role or both the creator and manager roles.

    If using the default accounts shipped with the Data Collector, you can use the admin login: admin / admin.
