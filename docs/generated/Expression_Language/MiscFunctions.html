Miscellaneous Functions

The expression language includes the following miscellaneous functions:

emptyList
Creates an empty list.
emptyMap
Creates an empty map.
runtime:conf("<runtime configuration name>")
Returns the value for the specified runtime configuration.
Uses the following argument:
  • runtime configuration name - Name of the configuration property to use. The property must defined in the sdc.properties file or in a runtime configuration file specified in the sdc.properties file.
sdc:id()
Returns the Data Collector ID.

When the Data Collector runs in standalone execution mode, the ID is a unique identifier associated with the Data Collector, such as 58efbb7c-faf4-4d8e-a056-f38667e325d0. The ID is stored in the following file: <SDCinstalldir>/data/sdc.id.

When the Data Collector runs in cluster mode, the ID is the Data Collector worker partition ID generated by Spark.
