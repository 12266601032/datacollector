Record Functions

Use record functions to get information about a record. The expression language provides the following record functions:

Type. Syntax: ${record:type("<field-path>')}
Returns the data type of a field.
Possible return values are:
  • Boolean
  • Char
  • Byte
  • Short
  • Integer
  • Long
  • Float
  • Double
  • Date
  • Datetime
  • Decimal
  • String
  • Byte-Array
  • Map
  • List
${record:value('<field-path>')}
Returns the value of the field in the appropriate data type.
${record:id()}
Returns the ID of the record generated by a stage, generally a source, upon record creation.
${record:creator()}
Returns the stage name that created the record, generally a source.
${record:path()}
Returns the stages in the pipeline that the record passed through.

Error Functions

Error functions provide information about bad records. The expression language provides the following error functions:

${error:stage()}
Returns the name of the stage that sent the record to error.
${error:code()}
Returns the error code for the bad record.
${error:message()}
Returns the error message for the bad record.
${error:collectorId()}
Returns the ID of the Data Collector that sent the record to error. Returns the format <hostname>:<port>.
${error:pipeline()}
The name of pipeline that sent the record to error.
${error:time()}
The time the record was sent to error.

String Functions

Use string functions to transform string data. The expression language provides the following string functions:

${str:substring(STRING, OFFSET, LEN)}
${str:trim(STRING)}
${str:toUpper(STRING)}
Converts string data to all capital letters.
${str:toLower(STRING)}
Converts string data to all lowercase letters.
${str:replaceChar(STRING, CHAR old, CHAR new)}
Replaces the character in a string with a new character.
${str:replaceString(STRING, STRING old, STRING new)}
Replaces a set of characters in a string with a new set of characters.
${str:truncate(STRING, INT len)}
Trucates the string to the specified length. Use an integer to specify the length.
${str:regexCapture(STRING, STRING pattern, INT group)}
${str:contains(STRING full, STRING subset)}
${str:startsWith(STRING full, STRING subset)}
${str:endWith(STRING full, STRING subset)}
${isIn(STRING full, STRING element)}
assumes 'full' is a list of comma separated values, looks for the element 'value'

assumes 'full' is a list of comma separated values, looks for the element 'value' to being in
