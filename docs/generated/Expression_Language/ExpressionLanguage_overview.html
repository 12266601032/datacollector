Expression Language

The expression language enables you to create expressions that evaluate or modify data. Use the expression language to create expressions in the Expression processor and conditions in the Stream Selector processor.

You can also use the expression language when you configure data rules, the Hadoop FS output file options, and Kafka Producer partitions.

You can use the following elements in an expression or condition:
  • Field names - Use the appropriate format to call a field name.
  • Constants
  • Literals
  • Operators
  • Functions
