Expression Language

The expression language enables you to create expressions that evaluate or modify data. Use the expression language to create expressions in the Expression processor and conditions in the Stream Selector processor.

Basic Syntax
Precede all expressions with a dollar sign and enclose them with curly brackets, as follows:
${<expression>}
For example, to add 2 +2, use the following syntax:${2+2}.
Calling Fields
To use the values in a field in an expression, use the following syntax:
record:value{"/<field_name>"}
For example, to add the sales of four quarters, you might use the following expression: 
${record:value("/Q1_Sales") + record:value("/Q2_Sales") + record:value("/Q3_Sales") + record:value("/Q4_Sales")}
Literals
Operators
You can use operators in expressions.
Constants
You can use constants in expressions. You can also define a constant variable. You can use the variable to represent a constant for any expression in the processor.
