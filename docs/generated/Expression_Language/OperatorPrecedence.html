Operator Precedence

The precedence of operators highest to lowest, left to right is as follows:

  • [ ]
  • ( ) - Used to change the precedence of operators
  • - (unary) not ! empty
  • * / div % mod
  • + - (binary)
  • < > <= >= lt gt le ge
  • == != eq ne
  • && and
  • || or
  • ? :
