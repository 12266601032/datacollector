Enabling Kerberos Authentication

You can use Kerberos authentication to connect to origin and destination systems, as well as YARN clusters.

By default, the Data Collector uses the user account who started it to connect to external systems. When you enable Kerberos, it can use the Kerberos principal to connect to external systems.

To enable Kerberos authentication, perform the following steps:
  1. Configure the Data Collector to use Kerberos.

    In the Data Collector configuration file, configure the Kerberos properties to enable Kerberos and define the principal and keytab.

  2. Configure the origin or destination stage to use Kerberos.
    In the pipeline, you can configure Kerberos authentication for the following stages:
    • Hadoop FS origin
    • Hadoop FS destination
    • HBase destination
Note: Cluster pipelines automatically use Kerberos authentication if the YARN cluster requires it, and if Kerberos is enabled for the Data Collector.
