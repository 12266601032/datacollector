Installing New Stages (Untar-Copy)

You can install a new stage by untarring the stage library and copying the

  1. Download the stage library and untar the file.
    For example:
    tar xvzf streamsets-datacollector-cdh_5_3_0-lib-1.0.0b1-SNAPSHOT.tar.gz
  2. Copy the contents of the stage-libs directory to the Data Collector stage-libs directory.
    For example:
    cp -r streamsets-datacollector-1.0.0b1-SNAPSHOT/stage-libs/streamsets-datacollector-cdh5_3_0-lib/ /tmp/streamsets-datacollector-1.0.0b1-SNAPSHOT/stage-libs/
  3. Restart the Data Collector.
