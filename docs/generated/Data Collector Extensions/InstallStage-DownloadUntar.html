Installing a Stage (Download-Untar)

You can install a stage by downloading the stage library to the appropriate directory and untarring it.

  1. Download the stage library to the parent directory of the Data Collector installation directory.
    For example, if the Data Collector is installed in the /opt directory, you would download the stage library to /opt.
  2. Untar the file.
    For example:
    tar xvzf streamsets-datacollector-cdh_5_3_0-lib-1.0.0b1-SNAPSHOT.tar.gz
  3. Stop and restart the Data Collector.
