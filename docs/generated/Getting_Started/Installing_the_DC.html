Installing and Launching the Data Collector (Tarball)

Install the Data Collector on the machine that you want to run a pipeline. After installation, you can launch the Data Collector.

  1. Download StreamSets SDC tarball and extract it to desired location.
  2. Optionally, set environment variables to point to appropriate directories:
    • SDC_CONF - SDC configuration directory.
    • SDC_DATA - SDC data directory, where pipeline state and configuration are stored.
    • SDC_LOG - SDC logging directory.
  3. Use the following command to launch the Data Collector and run it in the background: 
    $SDC_HOME/bin/sdc &
