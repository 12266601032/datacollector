Installing Additional Stages

The Data Collector includes a wide range of stages for pipeline development, but you might find a need for additional functionality.

At this time, you can install the following additional stages:
Additional Functionality Description

Cloudera CDH version 5.3.1

Cloudera CDH version 5.3.0

Cloudera CDH version 5.2.1

Use to write to the Cloudera distribution of Apache Hadoop.

Provides a Cloudera CDH destination for the installed version.
Kafka version 0.8.2.0

Kafka version 0.8.1.1

 Use to access Kafka clusters.

Provides a Kafka origin and destination for the installed version.

Installing a Stage (Download-Untar)

You can install a stage by downloading the stage library to the appropriate directory and untarring it.

  1. Download the stage library to the parent directory of the Data Collector installation directory.
    For example, if the Data Collector is installed in the /opt directory, you would download the stage library to /opt.
  2. Untar the file.
    For example:
    tar xvzf streamsets-datacollector-cdh_5_3_0-lib-1.0.0b1-SNAPSHOT.tar.gz
  3. Stop and restart the Data Collector.

Installing New Stages (Untar-Copy)

You can install a new stage by untarring the stage library and copying files.

  1. Download the stage library and untar the file.
    For example:
    tar xvzf streamsets-datacollector-cdh_5_3_0-lib-1.0.0b1-SNAPSHOT.tar.gz
  2. Copy the contents of the stage-libs directory to the Data Collector stage-libs directory.
    For example:
    cp -r streamsets-datacollector-1.0.0b1-SNAPSHOT/stage-libs/streamsets-datacollector-cdh5_3_0-lib/
   /tmp/streamsets-datacollector-1.0.0b1-SNAPSHOT/stage-libs/
  3. Restart the Data Collector.
