Additional Stage Libraries

The Data Collector includes a wide range of stages for pipeline development, but you might need additional functionality. You can install stage libraries to provide access to additional stages.

At this time, you can install the following stage libraries:
Stage Library Available Client Library Versions Description

0.8.2.x

Provides the Kafka Consumer and Kafka Producer.

 Use to write to the Cloudera CDH distribution of Hadoop.
Provides the following destinations:
  • Flume
  • Hadoop FS
  • HBase
  • Hive Streaming (5.4.x only)
  • Solr

Also provides the Hadoop FS origin for cluster mode pipelines.
CDH Cluster Kafka 1.2.x 5.4.x Use to read from the Cloudera distribution of the Apache Kafka 1.2.0.

Provides the Kafka Consumer origin for cluster mode pipelines.

CDH Kafka 1.2.0 (0.8.2.0) Use access the Cloudera distribution of Apache Kafka 1.2.0 (0.8.2.0).

Provides the Kafka Consumer origin and Kafka Producer destination.
