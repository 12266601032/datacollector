Additional Stage Libraries

The Data Collector includes a wide range of stages for pipeline development, but you might need additional functionality. You can install stage libraries to provide access to additional stages.

Installing Additional Stage Libraries

You can install a stage library by downloading the library and untarring it into the Data Collector installation directory.

  1. Download the stage library tar file to a directory in the machine where the Data Collector is installed.
  2. Untar the file to the Data Collector installation directory.
    For example, if you downloaded the stage library to your home directory and installed the Data Collector in the opt/local directory, you might use the following commands to untar a stage library:
    $ cd /opt/local
$ tar xvf ~/streamsets-datacollector-<stage_library>.tar.gz
  3. Restart the Data Collector.
The stages from the stage library become available in the Data Collector.
