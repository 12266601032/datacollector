Java Configuration Options

You can edit the Data Collector environment file, sdc-env.sh, to add Java configuration options. The Data Collector environment configuration file includes the following Java configuration options:
SDC_PRE_CLASSPATH
Defines the path to Java classes that you can use in custom stages such as the JavaScript or Jython Evaluator.
SDC_POST_CLASSPATH
Defines the path to Java classes that you can call from custom stages such as the JavaScript or Jython Evaluator.
SDC_JAVA_OPTS
Defines additional Java configuration options that you want the Data Collector to use. By default, this property includes the heap size for the Data Collector.
  • Java Heap Size Configuration

    You can define the Java heap size used by the Data Collector. By default, the Data Collector Java heap size is 1024 MB.
