Creating Multiple Instances

When you create an instance, you specify a directory. The Data Collector creates the directory and copies several directories to that location.
From the command line, enter the following command: 
create-sdc -home=<new SDC directory> (-httpPort=<SDC_HTTP_PORT> | -httpsPort=<SDC_HTTPS_PORT>)
Note: Enter a directory that does not exist. Also, configure the port number that you want to use. For more information about http and https, see HTTP Protocols.
