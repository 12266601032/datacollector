Random Port Number

You can configure the Data Collector to use a random port number. By default, the Data Collector uses a static port number defined in the Data Collector properties file.

Configure the Data Collector to use a random port when you want the operating system to supply a random port number or when you want to run the Data Collector in cluster mode.

To determine the port number and URL to use to access the Data Collector console, check the Data Collector log. For example, the following message indicates that the Data Collector is running on http://192.168.130.5:18630:
2015-06-06 10:41:07,764 DEBUG WebServerTask - Running on URI 'http://192.168.130.5:18630/', HTTPS 'false'
The Data Collector log is located in the following location:
<DataCollector_installdir>/log/sdc.log
