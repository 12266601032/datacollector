Multiple Data Collector Instances

You can create multiple Data Collector instances. Each instance uses the same executables, but can be configured differently and run different pipelines.

You might create multiple instances when you want to run several Data Collectors on the same host machine.

You can run the Data Collector instances at the same time, as long as the host machine has enough resources.

Creating Multiple Instances

When you create an instance, you specify a directory. The Data Collector creates the directory and copies several directories to that location.
  1. From the command line, enter the following command: 
    create-sdc -home=<new SDC directory> (-httpPort=<SDC_HTTP_PORT> | -httpsPort=<SDC_HTTPS_PORT>)
    Note: Enter a directory that does not exist. Also, configure the port number that you want to use. For more information about http and https, see HTTP Protocols.
    The Data Collector creates another Data Collector instance in the specified directory and installs
  2. To launch the new Data Collector, navigate to the new Data Collector directory and enter the following command: 
    sdc
    The new Data Collector starts. Use the http host and port number to access the new Data Collector.
