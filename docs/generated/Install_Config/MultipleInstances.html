Multiple Data Collector Instances

You can create multiple Data Collector instances. Each instance uses the same executables, but can be configured differently and run different pipelines.

You might create multiple instances when you want to run several Data Collectors on the same host machine.

You can run the Data Collector instances at the same time, as long as the host machine has enough resources.
