RPC Pipeline Types

To pass data across data centers, create two types of RPC pipelines:
origin pipeline
Processes data from the origin system and passes it to the destination RPC pipeline.
Uses an RPC destination to pass data to the destination pipeline. You can configure the RPC destination to write to multiple pipelines.
destination pipeline
Processes data from the origin pipeline and passes it to the destination system.
Uses an RPC origin to process data from the origin pipeline.
