Alerts

  • Metric Alerts

    Metric alerts are alerts based on pipeline and stage statistics, such as a high rate of error records or a batch of records taking too long to process.
  • Data Rules and Alerts

    Data rules define the information that you want to see about the data that passes between stages. You can create data rules based on any link in the pipeline. You can also create alerts based on the data rules.
  • Configuring Email for Alerts

    You can define the email addresses to receive metric and data alerts. When an alert triggers an email, the Data Collector sends an email to every address in the list.
