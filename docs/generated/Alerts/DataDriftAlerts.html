Data Drift Alerts

You can create data alerts to indicate when changes to the structure of data changes, a.k.a. data drift.

The expression language provides data drift functions for use in data drift alerts. You can create alerts for the following types of data drift:
  • Field name changes - Use the drift:name() function to trigger an alert when field names change.
  • Field order changes - Use the drift:order() function to trigger an alert when the field order changes.
  • Number of fields in the record - Use the drift:size() function to trigger an alert when the number of fields changes in a record.
  • Data type of a field changes - Use the drift:type() function to trigger an alert when the data type of a specified field changes.
