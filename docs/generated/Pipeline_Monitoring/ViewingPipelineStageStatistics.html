Viewing Pipeline and Stage Statistics

When you monitor a pipeline, you can view real-time summary and error statistics for the pipeline and for stages in the pipeline. You can view the configuration of the pipeline or selected stage to help determine if you need to make changes. You can also view the run history for the pipeline.

By default, the Data Collector window displays pipeline monitoring information when it runs a pipeline. When you go to the Data Collector window, the Monitor panel displays pipeline summary information by default. You can use tabs in the Monitor panel to view additional information.

The following table lists the monitoring information available on each tab:
Icon Tab Information
Summary

Displays the record count for the pipeline, record and batch throughput for the pipeline or individual stages, and batch processing statistics for the pipeline or individual stages.

Tip: You can hover over different parts of the charts to view exact numbers.
Error For a pipeline, displays histograms for the number of error records by five minute decay and the number of error records by stage since the pipeline started.

For a stage, displays the number of error records and the number of stage errors.

Configuration Displays configuration information for the pipeline or selected stage. You cannot configure the pipeline when it is running.
History Displays the status and history of the pipeline.
