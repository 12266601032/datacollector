Metric Types

Metric alerts provide different types of alerts based on the metric type that you select.

For example, when you select a counter type, you can create alerts based on the number of error records generated by a specific stage.

Metric alerts include the following metric types:
Counter
Provides metric alerts based on a count of different record types and of stage errors for any stage in the pipeline.
Histogram
Provides metric alerts based on a histogram of different record types and stage errors for any stage. You can also use batch-based histograms.
Meter
Provides metric alerts based on a rates of different record types and stage errors for any stage. You can also use batch-based meters.
Timer
Provides metric alerts based on batch processing timers for the pipeline or any stage in the pipeline.
