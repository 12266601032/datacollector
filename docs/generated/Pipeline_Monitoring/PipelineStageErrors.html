Pipeline and Stage Errors

When you monitor a pipeline, you can view error statistics for the pipeline and each stage. You can also view a sampling of the error records.

The Rules tab in the Monitor panel displays pipeline errors by default:

To view error statistics for a stage, click the stage. The Rules tab of the Monitor panel displays a sample of error records with related error messages, as well as the count and a histogram of errors:

The number of error records saved in memory is defined in the Data Collector properties file.
