Pipeline Monitoring

When the Data Collector runs a pipeline, you can view real-time statistics about the pipeline, examine a sample of the data being processed, and create rules and alerts.

When you access the Data Collector console as the Data Collector runs a pipeline, the console displays the pipeline in Monitor mode. In Monitor mode, you can perform the following tasks:
  • Take and review a snapshot of data
  • View stage and pipeline statistics
