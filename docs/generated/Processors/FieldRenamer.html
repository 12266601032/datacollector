Configuring a Field Renamer

Information forthcoming...

You can access the latest documentation for this release on the StreamSets website: https://www.streamsets.com/documentation/1.1.0/DCUserGuide/help/index.html.

You can also configure the Data Collector to use the latest documentation as the context-sensitive help. This requires an internet connection and a login.

  1. In the top right corner of the Data Collector console, click Help > Settings.
  2. For the Help property, select Hosted Help, and then click Done.
    When you click the context-sensitive help icon, the Data Collector launches hosted help with the latest documentation after you log in.
