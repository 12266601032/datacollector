Geo IP

The Geo IP processor provides geographic location information for IP addresses. Geo IP uses the Maxmind GeoIP2 database to provide geolocation information.

To use the Geo IP processor, you specify one or more IP address input fields, name the corresponding output field, and specify the geolocation information that you want.

Geo IP can provide the following geolocation information for each IP address.
  • Country name
  • Country ISO code
  • City name
  • Latitude
  • Longitude
