Field Masker

The Field Masker processor masks string values based on the selected mask type. You can use variable-length, fixed-length, or custom masks. Custom masks can reveal part of the string value.

Use the Field Masker to mask sensitive string data. For example, you might use a custom mask to mask the last four digits of a phone number.

Tip: To mask non-string data, you might use the Field Converter processor to convert non-string data to String, then pass the data to the Field Masker.

Mask Types

You can use the following mask types to mask data:

Fixed-length
Replaces values with a fixed-length mask. Use when you want to mask any variability in length in the data.
The following example uses a fixed-length masks to hide passwords:
Original Password 
1234
donKey
022367snowfall
 
Fixed-Length Mask
asd302kd0
2v03msO3d
L92m1sN3q
Variable-length
Replaces values with a variable-length mask. Use when you want to reveal the lengths of the original data.
The following example uses a variable length mask to hide the same passwords:
Original Password (original)
1234
donKey
022367snowfall
 
Variable-Length Mask
asd3
2v03ms
L92m1sN3q0jaOmE67Ws
Custom
Replaces values with a mask based on a user-defined pattern.
When you define the pattern for the mask, you can use a hash mark (#) to reveal the character in that location of the hash mark. All other characters are used as constants in the mask. For example, the following mask pattern reveals the area code of a phone number and replaces all other digits in the phone number with an x: ###-xxx-xxxx.
Tip: To avoid confusing masking characters for real data, use one masking character instead of a mix of masking characters.

Configuring a Field Masker

Configure a Field Masker to mask sensitive data.

  1. In the Properties panel, optionally enter a name and description for the stage.
  2. For Required Fields, enter any fields that you want to require data.
    You might include any field that the processor uses.
  3. On the Mask tab, configure the following properties:
    Field Masker Configuration Property Description
    Fields to Mask One or more String fields to mask with the same mask type.
    Mask Type Mask type to hide field values. Select one of the following options:
    • Fixed-length - Replaces values with a fixed-length mask.
    • Variable-length - Replaces values with a mask the length of the original value.
    • Custom - Replaces values with a user-defined mask.
    Custom Mask Mask pattern for a custom mask. Enter the pattern that you want to use.

    Use the hash mark (#) to display characters in the specified location. Use any other character as a masking character.
  4. To mask another field, click the Add icon, and then repeat the previous step.
