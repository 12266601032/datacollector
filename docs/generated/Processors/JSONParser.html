JSON Parser

The JSON Parser parses a JSON object embedded in String field and passes the resulting data into the pipeline. Based on the contents of the JSON object, the resulting field is either a string, a map, or an array.

Configuring a JSON Parser

Use a JSON Parser to parse a JSON object in a String field.

When you configure a JSON Parser, you specify the field to parse and the output field to use.
  1. In the Properties panel, optionally enter a name and description for the stage.
  2. For Required Fields, enter any fields that you want to require data.
    You might include any field that the processor uses.
  3. On the Parse tab, configure the following properties:
    JSON Parsing Property Description
    Field to Parse Field that contains the JSON object.
    New Parsed Field Output field for the parsed JSON object.
    Error Record Determines what to do with error records:
    • Discard - Discards the record.
    • Send to Error - Sends the record to the pipeline for error handling.
    • Stop Pipeline - Stops the pipeline.
