JavaScript Evaluator

The JavaScript Evaluator processes data based on custom JavaScript. Use the JavaScript processor to incorporate custom JavaScript script in a pipeline.

The JavaScript Evaluator can process records as follows:

Record by Record
Processes each record individually.
When processing records individually, the processor handles error records based on the Error Record property defined for the processor.
Record Batch
Processes batches of records.
When processing records by the batch, the script must include error handling code.

Configuring a JavaScript Evaluator

Configure a JavaScript Evaluator to use custom JavaScript in a pipeline.

On the JavaScript tab, configure the following properties:
JavaScript Property Description
Record Processing Mode Determines how the JavaScript Evaluator processes data:
  • Record by Record - Processes records individually. Performs error handling.
  • Record Batch - Processes records in batches. Requires error handling code in the Jython script.
Script JavaScript code to use in the processor.
