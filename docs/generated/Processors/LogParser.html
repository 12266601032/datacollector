Log Parser

The Log Parser processor generates a record for every log line and parses log lines based on the specified log format.

When you configure the Log Parser, you define the text field that contains the log line and the location for parsed fields.

If the record contains fields in addition to the text field to be parsed, those fields are passed through by default. Parsed fields are written to the specified location, overwriting any existing data.

  • Log Formats

    When you use an origin to read log data, you define the format of the log files to be read.
