Field Converter

The Field Converter converts the datatype of a field to a compatible datatype. You might use the Field Converter to convert the data type of a field before performing calculations.

When you convert a field, configure the conversion for an appropriate, compatible datatype. Consider the actual data in the field as well, since even valid conversions can cause truncation of data.

For example, converting a field from Integer to Decimal is valid. Converting a field from Decimal to Integer is also valid, but the conversion can cause truncation of any decimal values in the data.

Tip: You can use data preview to verify the data in the fields.
When you convert a String field to Date or Datetime, you specify the date or datetime format that you want to use. You can use any valid format.

Valid Type Conversions

The following table lists the data types that can be converted to another data type. Byte Array, List, and Map data types cannot be converted.

Target Data Type Source Data Type
Boolean Byte, Decimal, Double, Float, Integer, Long, Short, String
Byte Decimal, Double, Float, Integer, Long, Short, String
Character String
Date Long, String
Decimal Byte, Double, Float, Integer, Long, Short, String
Double Byte, Decimal, Integer, Float, Long, Short, String
Float Byte, Decimal, Double, Integer, Long, Short, String
Integer Byte, Decimal, Double, Float, Long, Short, String
Long Byte, Decimal, Double, Float, Integer, Short, String
Short Byte, Decimal, Double, Float, Integer, Long, String
String All supported data types.

Configuring a Field Converter

Configure a Field Converter to convert the data type of a field.

  1. In the Properties panel, optionally enter a name and description for the stage.
  2. For Required Fields, enter any fields that you want to require data.
    You might include any field that the processor uses.
  3. On the Conversions tab, configure the following properties:
    Field Converter Configuration Property Description
    Fields to Convert One or more field to convert to the same data type.
    Convert to Type Data type for conversion.
    Data Locale Locale of the data in the field. Can determine how the Field Converter formats converted data, such as a comma or period as a decimal separator.
    Date Format Format for a Date or Datetime field. Select or enter any valid format.
  4. To convert another field, click the Add icon and then repeat the previous step.
