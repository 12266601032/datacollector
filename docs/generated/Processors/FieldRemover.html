Field Remover

The Field Remover removes fields from the pipeline. Use the Field Remover to discard data that you do not need in the pipeline.

Configuring a Field Remover

Use a Field Remover to remove unwanted fields from the pipeline.

When you configure the Field Remover, you can list the fields that you want to keep so the Field Remover removes the remaining fields. Or, you can list the fields that you want to remove.
  1. In the Properties panel, optionally enter a name and description for the stage.
  2. For Required Fields, enter any fields that you want to require data.
    You might include any field that the processor uses.
  3. On the Remove tab, configure the following properties:
    Field Remover Property Description
    Action Indicates whether to keep or drop the listed fields.
    Fields List of fields to keep or remove.
