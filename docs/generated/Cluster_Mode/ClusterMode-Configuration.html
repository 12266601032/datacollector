Cluster Mode Configuration

To run the Data Collector in cluster mode, perform the following steps:
  1. Install the Data Collector on a Spark on YARN gateway node.
  2. In the Data Collector properties file, set the sdc.execution.mode property to cluster.
  3. In the pipeline properties, on the General tab, set the Execution Mode property to Cluster.
  4. Use a Kafka Consumer origin in the pipeline.
