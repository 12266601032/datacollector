Cluster Mode

You can run the Data Collector in cluster mode to process large volumes of data from a Kafka cluster.

When a Data Collector runs in cluster mode, it runs as an application within Spark. Spark is an open source cluster-computing application that is installed as part of the Kafka CDH.

Within Spark, the Data Collector spawns a Data Collector instance for each partition in the Kafka cluster. So each partition has a Data Collector instance to process data.

In cluster mode, you can preview data and monitor each Data Collector in the cluster.

Note: If you add a partition to the topic being read by the Data Collector, restart the pipeline to enable the Data Collector to read from the new partition.
