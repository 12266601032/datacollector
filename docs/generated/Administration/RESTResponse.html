REST Response

You can view REST response JSON data for different aspects of the Data Collector, such as pipeline configuration information or monitoring details.

You can use the REST response information to provide Data Collector details to a REST-based monitoring system. Or you might use the information in conjunction with the Data Collector REST API.

You can access REST response information from the location where the relevant information displays. For example, you can download the JVM Metrics REST response from the JVM Metrics page.

You can access the following REST response data:
  • Pipeline Configuration - Provides information about the pipeline and each stage in the pipeline.
  • Pipeline Rules - Provides information about metric and data rules and alerts.
  • Definitions - Provides information about all available Data Collector stages.
  • Pipeline Preview - Provides information about the preview data moving through the pipeline. Also includes monitoring information that is not used in preview.
  • Pipeline Monitoring - Provides monitoring information for the pipeline.
  • Pipeline Status - Provides the current status of the pipeline.
  • JVM Metrics - Provides JVM metrics for the Data Collector.

You can configure the Data Collector to hide the display of REST responses. Click Help > Settings, and then click Hide the REST Response Menu.

Accessing a REST Response

You can access REST response data from the following locations:
Edit mode
From the Properties panel, you can use the More icon to access the following REST responses:
  • Pipeline Configuration
  • Pipeline Rules
  • Definitions
Preview mode
From the Preview panel, you can use the More icon access the pipeline preview REST response.
Monitor mode
From the Monitor panel, you can use the More icon to access the following REST responses:
  • Pipeline Configuration
  • Pipeline Rules
  • Definitions
  • Pipeline Monitoring
  • Pipeline Status
JVM Metrics page
From the JVM Metrics page, Administration > JVM Metrics , you can use the Settings icon to access the JVM metrics REST response.
