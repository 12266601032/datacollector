REST Response

You can view the REST response that the Data Collector console uses to generate displays. The REST response is JSON data that you can send to StreamSets client services to help troubleshoot issues.

You can view the following types of REST responses:
  • Pipeline Configuration - Provides information about the pipeline and each stage in the pipeline.
  • Pipeline Rules - Provides information about metric and data rules and alerts.
  • Definitions - Provides information about all available Data Collector stages.
  • Pipeline Preview - Provides information about the preview data moving through the pipeline. Also includes monitoring information that is not used in preview.
  • Pipeline Monitoring - Provides monitoring information for the pipeline.
  • Pipeline Status - Provides the current status of the pipeline.
  • JVM Metrics - Provides JVM metrics for the Data Collector.
