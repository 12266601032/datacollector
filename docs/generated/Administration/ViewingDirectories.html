Viewing Data Collector Directories

You can view the directories that the Data Collector uses. You might check the directories being used to access a file in the directory or to increase the amount of available space for a directory.

To view Data Collector directories, click Administration > SDC Directories.

The following table describes the Data Collector directories that display:
Directory Includes Default Location
Run Time Base directory for Data Collector executables and related files. <SDCinstalldir>
Configuration The Data Collector configuration file (sdc.properties), and related help realm properties files and keystore files.

Also includes the logj4 properties file.

 <SDCinstalldir>/etc
Data Pipeline configuration and run details. <SDCinstalldir>/data
Log Data Collector log file, sdc.log. <SDCinstalldir>/log
Resources Directory for runtime resource files. <SDCinstalldir>/resources
