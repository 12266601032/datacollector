Data Collector Console - Preview Mode

In Preview mode, you can use the Data Collector console to view how data passes through the pipeline.

The following image shows the Data Collector console in Preview mode:

Area / Icon Name Description
1 Pipeline canvas Highlights the previewed stage. You can select a stage to view its data preview.
2 Preview panel Displays input and output data.
3 Previewing Stage / Multiple Stages Use to select the type of preview you want to perform. You can preview a single stage or multiple stages.
Previous Stage icon Use to preview the stage to the left of the selected stage.
Next Stage icon Use to preview the stage to the right of the selected stage.
Re-run Stage icon Use to run edited input data through the stage so you can review the results.
Delete icon Use to delete a record from the data preview.

Preview Codes

When you preview data, the Preview panel uses different colors to represent different types of data. The Preview panel also uses colors, italics, and asterisks to highlight the field values and records that you have changed for testing.

The following table describes the color and asterisk coding:
Preview Code Description
Black values Date data
Blue values Numeric data
Green values String data
Red values Boolean data
Asterisk Records that include edited field values
Red italic field labels Fields that contain edited data
Italic values Edited data
Green stage First stage in a multiple-stage preview
Red stage Last stage in a multiple-stage preview
