Previewing a Single Stage

You can preview data for a single stage. In the Preview panel, you can drill down to review the values for each record to determine if the stage transforms data as expected. You can also edit the preview data to see how the stage processes your changes.

  1. From the pipeline canvas, click Preview.
    If the Preview icon is disabled, check the Issues list for unconnected stages and required properties that are not defined.
    The Preview panel highlights the origin stage and displays preview records in the Output Data column. Since this is the origin of the pipeline, no input data displays.

    You can drill down to view the details of a record.

  2. To delete a record that you do not want to use, click the Delete icon.
  3. To view data for the next stage, click the Next Stage icon. Or, to view data for a different stage, select the stage in the pipeline canvas.
  4. To change field values, in the Input Data column, click the value that you want to change and enter a new value.
    You can edit values for any input data.
  5. To process changed data, click the Run Preview With Changes icon.
    In the Input Data column, records with changed values display with an asterisk and the changed values are highlighted. The Output Data column displays the results of processing. You can change and process preview data as often as necessary.
  6. To view the changed data in other stages, use the stage navigation icons or select the stage to review.
  7. To revert all changes and return to the beginning of the data preview, click the Revert Changes icon.
  8. To preview the next set of data, click Refresh Preview.
  9. To exit data preview and return to pipeline configuration, click Close Preview.
