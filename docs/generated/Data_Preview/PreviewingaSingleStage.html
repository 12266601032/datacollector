Previewing and Tuning a Single Stage

You can preview data for a single stage. When you preview data for a single stage, you can also change the data to test stage processing. Preview data to verify that the stage processes data as expected.

In the Preview panel, you can drill down to review the values for each record to determine if the stage transforms data as expected.

You can also change record values to view how the stage processes the changed data. Use this functionality to fine-tune stage configuration. For example, if the stage filters integer data based on an expression, you might change the input data to test positive and negative integer values, as well as zero.

You can change values in the following locations:
  • The output data column for an origin stage.
  • The input data column for all other stages.

When you change preview data, you can pass the changed data through the pipeline, or you can revert your changes to return to the original data.

  1. From the pipeline canvas, click Preview.
    If the Preview button is disabled, the required fields for the origin stage are not configured.
    The canvas centers on the origin stage. The Preview panel displays preview data in the Output Data column. Since this is the origin of the pipeline, no input data displays.

    You can drill down to view the details of a record.

  2. To delete a record that you do not want to use in the preview, click the Delete icon.
  3. To view data for the next stage, click Next Stage. Or, to view data for a different stage, select the stage in the pipeline canvas.
  4. To change field values to test stage processing, in the Input Data column, click the value that you want to change and enter a new value.

    You can edit values for any input data.
  5. To process changed data, click Re-run Stage.
    In the Input Data column, records with changed values display with an asterisk and the changed values are highlighted. The Output Data column displays the results of the stage processing. You can change and process preview data as often as necessary.
  6. To use the changed data in other downstream stages, click Next Stage. To revert all changes and return to the beginning of the data preview, click Revert Changes.
  7. To preview the next set of data, click Refresh Preview.
  8. To exit data preview and return to pipeline configuration, click Close Preview.
