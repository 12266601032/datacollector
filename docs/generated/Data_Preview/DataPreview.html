Data Preview

You can preview data to help build or fine-tune a pipeline. When you preview data, the Data Collector reads records from the origin and passes it through the pipeline. You can examine the data at each stage to verify how the pipeline processes data.

You can preview data after you configure the origin stage.

When you preview data, the Preview panel displays the input and output data for the selected stage. You can systematically preview data for each stage in the pipeline. When you preview a single stage, you can edit preview data to perform detailed testing of the stage.

You can also preview the input and output data for a set of stages to verify how the set of stages processes data.

You can refresh the data preview to review the pipeline with another set of data.

Data Collector Window - Preview Mode

In Preview mode, you can use the Data Collector window to view how data passes through the pipeline.

Preview Colors and Asterisks

When you preview data, the Preview panel uses different colors to represent different types of data. The Preview panel also uses colors and asterisks to highlight the field values and records that you have changed for testing.

The following table describes the color and asterisk coding:
Preview Code Description
Black values Date data
Blue values Numeric data
Green values String data
Red values Boolean data
Red field labels Fields that contain changed data
Asterisk Records that include changed field values
Green stage First stage in a multiple-stage preview
Red stage Last stage in a multiple-stage preview
