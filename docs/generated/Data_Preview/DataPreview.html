Data Preview

You can preview data to help build or fine-tune a pipeline. When you preview data, the Data Collector reads records from the origin and passes it through the pipeline. You can examine input and output data for one stage at a time or for a group of stages.

When you preview data for a single stage, the Preview panel displays the input and output data for the selected stage. You can select any stage to view the input and output data for the stage.

You can also preview data for a group of stages to verify how the group processes data.

When you preview data, you can edit the data and pass it through the pipeline, and then examine the results.

You can also refresh the preview to use a new set of data from the origin.

Availability

You can preview complete and incomplete pipelines. The Data Preview icon becomes active when data preview is available.

You can preview data under the following conditions:
  • All stages in the pipeline are connected
  • All required properties are defined
Tip: Stage configuration does not have to be accurate or complete to preview data. After you connect all stages, you can enable data preview by entering any valid value for required properties.

Editing Preview Data

You can edit preview data to view how a stage or group of stages processes the changed data. Use this functionality to fine-tune pipeline configuration.

For example, when the stage filters integer data based on an expression, you might change the input data to test positive and negative integer values, as well as zero.

You can change values in the following locations:
  • The output data column for an origin stage.
  • The input data column for all other stages.

When you edit preview data, you can pass the changed data through the pipeline, or you can revert your changes to return to the original data.
