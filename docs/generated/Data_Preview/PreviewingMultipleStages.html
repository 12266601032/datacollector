Previewing Multiple Stages

You can preview data for a group of linked stages within a pipeline. Preview multiple stages to verify that the group of stages processes data as expected.

When you preview multiple stages, you select the first stage and the last stage in the group. The Preview panel then displays the input data and the output data for the group. The input data is the data that enters the first stage. The output data is the data that passes from the last stage.

In the Preview panel, you can drill down to review the values for each record to determine if the group of stages transforms data as expected.

  1. From the pipeline canvas, click Preview.
    If the Preview button is disabled, check the Issues list for unconnected stages and required properties that are not defined.
    The Preview panel highlights the origin stage and displays preview records in the Output Data column. Since this is the origin of the pipeline, no input data displays.

    You can drill down to view the details of a record.

  2. To delete a record that you do not want to use, click the Delete icon.
  3. To preview multiple stages, click Multiple.
    The Preview panel displays two lists of stages.
  4. From the list on the left, select the first stage to use.
  5. In the list on the right, select the last stage to use.
    The Preview panel displays the input and output data for the group of stages. You can drill down to review the details of each record.
  6. To change field values, in the Input Data column, click the value that you want to change and enter a new value.
    You can edit values for any input data.
  7. To process changed data, click the Re-Run Preview With Changes icon.
    In the Input Data column, records with changed values display with an asterisk and the changed values are highlighted. The Output Data column displays the results of processing. You can change and process preview data as often as necessary.
  8. To revert all changes, click the Revert Changes icon.
  9. To preview the next set of data, click Refresh Preview.
  10. To return to previewing a single stage, click Single.
  11. To exit data preview and return to pipeline configuration, click Close Preview.
