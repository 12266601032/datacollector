Previewing Multiple Stages

You can preview data for a set of linked stages within a pipeline. Preview multiple stages to verify that the set of stages processes data as expected.

When you preview multiple stages, you select the first stage and the last stage in the set. The Preview panel then displays the input data and the output data for the set of stages. The input data is the data that enters the first stage in the set. The output data is the data that exits the last stage in the set.

In the Preview panel, you can drill down to review the values for each record to determine if the set of stages transforms data as expected.

  1. From the pipeline canvas, click Preview.
    If the Preview button is disabled, the required fields for the origin stage are not configured.
    The canvas centers on the origin stage. The Preview panel displays preview data in the Output Data column. Since this is the origin of the pipeline, no input data displays.

    You can drill down to view the details of a record.

  2. To delete a record that you do not want to use in the preview, click the Delete icon.
  3. To preview multiple stages, click Preview Stage > Preview Multiple Stages.
    The Preview panel displays two lists of stages.
  4. From the First list, select the first stage to include.
  5. In the Last list, select the last stage to include.
    The Preview panel displays the input and output data for the set of stages. You can drill down to review the details of each record.
  6. To preview the next set of data, click Refresh Preview.
  7. To return to previewing a single stage, click Preview Stage > Preview Single Stage.
  8. To exit data preview and return to pipeline configuration, click Close Preview.
