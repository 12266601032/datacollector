Defining Pipeline Constants

Define a pipeline constant to represent a constant that you want to reuse within the pipeline or to update easily.

You can enter any text as the value for a pipeline constant, but the value that you enter is treated as a literal constant.

  1. To define a pipeline constant, in the Pipeline properties panel, click the Constants tab.
  2. Enter a name and value for the pipeline constant.
  3. To create additional pipeline constants, click Add.
