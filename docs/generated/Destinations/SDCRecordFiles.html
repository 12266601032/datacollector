SDC Record Files

Use the SDC Record Files destination to write Data Collector records to a local file system. Data Collector records are JSON files of a specific format.

Use SDC Record Files to create Data Collector record files, such as error record files, that you want to easily reprocess. You can use the Directory origin with the Data Collector Records data format to process records written by SDC Record Files.

You can also use SDC Record Files when you want to write output records in JSON format to a local file system.

Configuring an SDC Record Files Destination

Configure the SDC Record File destination to write Data Collector JSON records to a local file system.

  1. In the Properties panel, optionally enter a name and description for the stage.
  2. Click the Files tab and configure the following properties:
    Local File System Property Description
    Directory Local directory for the output files.
    File Wait Time (secs) Amount of time to wait for records. After the configured time, SDC Record Files creates a new output file.
    Max File Size (MB) Maximum size for output files. Upon reaching the limit, SDC Record Files creates a new output file. Use 0 to avoid using this property.
