SDC Record Files

Use the SDC Record Files destination to use the SDC Record data format to write files to a local file system.

Use SDC Record Files to create output files that can be easily reprocessed by another Data Collector pipeline. You can use the Directory and Kafka Consumer origins to process records written by SDC Record Files.

Configuring an SDC Record Files Destination

Configure the SDC Record File destination to write SDC Record files to a local file system.

  1. In the Properties panel, optionally enter a name and description for the stage.
  2. Click the Files tab and configure the following properties:
    SDC Record Files Property Description
    Directory Local directory for the output files.
    File Wait Time (secs) Amount of time to wait for records. After the configured time, SDC Record Files creates a new output file.
    Max File Size (MB) Maximum size for output files. Upon reaching the limit, SDC Record Files creates a new output file.

    Use 0 to opt out of this property.
