Hadoop FS

The Hadoop FS destination writes data to a Hadoop file system. You can write delimited or JSON data to flat files or Hadoop sequence files.

  • Output Directory Generation

    Hadoop FS creates directories based on a time-based template that you define and the time basis you choose. It writes records to the resulting directories based on the values of the record.
