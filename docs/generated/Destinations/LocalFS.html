Local File System

Use the Local File System destination to write records to files in a local file system.

Configuring a Local File System Destination

Configure the Local File System destination to write records to a local file system.

  1. In the Properties panel, optionally enter a name and description for the stage.
  2. For Required Fields, enter any fields that you want to require data.
    You might include any field that the processor uses.
  3. Click the Files tab and configure the following properties:
    Local File System Property Description
    Directory Local directory for the output files.
    File Wait Time Amount of time to wait for records. After the configured time, Local File System creates a new output file.
