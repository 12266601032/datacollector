Local FS

Use the Local File System destination to write records to files in a local file system.

Configuring a Local FS Destination

Configure the Local FS destination to write records to a local file system.

Click the Files tab and configure the following properties:
Local File System Property Description
Directory Local directory for the output files.
File Wait Time Amount of time to wait for records. After the configured time, Local File System creates a new output file.
