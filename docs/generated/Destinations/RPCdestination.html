RPC

= The RPC destination writes data over the WAN to an RPC origin. Use the RPC destination as part of an RPC origin pipeline that passes data to an RPC origin in the destination pipeline.

When you configure the RPC destination, you specify the RPC ID and RPC connections to use.

Related concepts

RPC Pipelines

RPC Connections

When you configure an RPC destination, you configure RPC connections to define where to pass data.

The RPC connection is the host and port number of the RPC origin to receive the data. You can pass data to multiple RPC origins to allow processing of data by multiple destination pipelines.
