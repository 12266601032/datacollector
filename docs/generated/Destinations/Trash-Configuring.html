Configuring a Trash Destination

Use a Trash destination as a placeholder for testing or to gather statistics or receive alerts about discarded records.

You can configure required fields for the Trash destination. When a record does not include data in the required fields, it is written to the error log.

  1. In the Properties panel, optionally enter a name and description for the stage.
  2. For Required Fields, enter any fields that you want to require data.
