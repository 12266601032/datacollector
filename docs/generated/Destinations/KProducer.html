Kafka Producer

The Kafka Producer passes data to a Kafka cluster. The Kakfa Producer can pass data to Kafka as delimited, text, or JSON data.

When you configure a Kafka Producer, you configure the brokers, topic, partition strategy, and payload type to use. You can add additional Kafka configuration properties as needed.

The Kafka Producer passes data to partitions in the Kafka topic based on the partition strategy that you choose.

Writing Text and Delimited Data

When you write delimited data to Kafka, you define the fields to pass data to the cluster. When you write text data, you define a single field to pass the data.

When necessary, use an Expression Evaluator to merge data into the fields that you want to use.

Broker List

The Kafka Producer connects to Kafka based on the topic and associated brokers that you specify. To ensure a connection in case a specified broker goes down, list as many brokers as possible.

Additional Kafka Properties

You can add custom Kafka configuration properties to the Kafka Producer.

When you add the Kafka configuration property, enter the Kafka property name and the value. The Kafka Consumer does not validate the property names or values.

Note: The Kafka Producer uses the broker.host Kafka configuration property. The Kafka Consumer ignores user-defined values for the broker.list property.
