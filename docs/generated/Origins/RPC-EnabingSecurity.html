Enabling SSL

You can use HTTPS to securely transfer data to the RPC origin. To use HTTPS you need a valid keystore file and password.

To use HTTPS for incoming data:
  1. Save the keystore file in the Data Collector resources directory: <SDCinstalldir>/resources.
  2. In the RPC origin, enter the filename and password.
