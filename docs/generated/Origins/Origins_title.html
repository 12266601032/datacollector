Origins

  • Origins

    An origin stage represents the data that you want the pipeline to process. You can use a single origin stage in a pipeline.
  • Previewing Raw Source Data

    You can preview raw source data when you configure an origin. Preview raw source data when reviewing the data might help with origin configuration.
  • Directory

    The Directory origin reads data from files in a directory.
  • File Tail

    The File Tail origin reads data as it is written to a file. As long as the pipeline runs, File Tail captures log or JSON data that is written to the file. The File Tail origin does not read data already in the file or data that is written to the file when the pipeline is not running.
  • Kafka Consumer

    The Kafka Consumer reads data from a Kafka cluster and processes the data based on the type of data stored in Kafka. You can use the Kafka Consumer to read log, delimited, JSON, and XML data from a Kafka cluster.
