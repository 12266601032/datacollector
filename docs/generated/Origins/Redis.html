Redis Consumer

The Redis Consumer origin reads messages from Redis.

When you configure Redis Consumer, you define Redis URI and the channels to read from. You can specify specific channels or provide patterns to subscribe to all matching channels. You can also configure the connection timeout.

    You can specify channels and patterns to define the data that the Redis Consumer origin processes.
