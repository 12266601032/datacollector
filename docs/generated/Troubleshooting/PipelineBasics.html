Pipeline Basics

Use the following tips to help you with pipeline basics.

When I go to the Data Collector console, I get a "Webpage not available" error message.
The Data Collector is not running. Start the Data Collector.
Why isn't the Start icon enabled?
You can start a pipeline when it is valid. Use the Issues icon to review the list of issues in your pipeline. When you resolve the issues, the Start icon becomes enabled.
Why doesn't the Select Fields with Preview Data option work? No preview data displays.
Select Fields with Preview Data can provide preview data when the pipeline is valid for data preview. Make sure all stages are connected. You might also need to enter temporary values for required properties to enable data preview.

Data Preview

Why isn't the Data Preview icon enabled?
You can preview data after you connect all stages in the pipeline and configure required properties. You can use any valid value as a placeholder.
Why doesn't the data preview show any data?
If data preview doesn't show any data, one of the following issues might have occurred:
  • The origin might not be configured correctly. In the Preview panel, check the Issues tab of the origin stage for related issues.
  • The origin might not have any data at the moment.
Why can't I edit preview data?
When you can edit preview data in the following locations in a pipeline.
  • The output data column for an origin.
  • The input data column for processors.
Edit preview data to test or debug pipeline processing.
