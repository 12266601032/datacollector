Importing a Pipeline

Import a pipeline to use a pipeline developed on a different Data Collector. When you import a pipeline, you create a pipeline and then import the pipeline.

Import a pipeline from a pipeline file. Pipeline files are JSON files exported from another Data Collector.
  1. In the Pipeline library, click Create New Pipeline.
  2. In the New Pipeline dialog box, enter a pipeline name and optional description. Click Save.
  3. In the Pipeline library, click the Actions icon of the new pipeline, and then click Import.
  4. In the Import Pipeline window, browse to locate the pipeline file and click Import.
    The pipeline displays in the canvas.
