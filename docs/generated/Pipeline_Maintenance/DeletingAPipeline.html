Deleting a Pipeline

You can delete a pipeline when you no longer need it. Deleting a pipeline is permanent. If you think you might want to archive a pipeline, you might export the pipeline before you delete it.

  1. In the Pipeline library, click the Actions icon for the pipeline.
  2. Click Delete.
    A confirmation window appears.
  3. To delete the pipeline, click Yes.
