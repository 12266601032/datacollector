Duplicating a Pipeline

Duplicate a pipeline when you want to keep the existing version of a pipeline while continuing to configure a duplicate version. A duplicate is an exact copy of the original pipeline.

When you duplicate a pipeline, you can rename the pipeline.

  1. In the Pipeline library, click the Actions icon for the pipeline.
  2. Click Duplicate.
  3. In the Duplicate Pipeline Definition dialog box, enter a name for the duplicate pipeline and click Duplicate.
    The pipeline displays in the canvas.
