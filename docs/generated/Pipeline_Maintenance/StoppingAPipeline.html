Stopping a Pipeline

Stop a pipeline when you want the Data Collector to stop processing data for the pipeline.

When the Data Collector runs a pipeline, it displays in the Data Collector console in Monitor mode by default.

  1. Verify that the pipeline that you want to stop appears in the pipeline canvas.
    If it does not, select the pipeline that you want to stop from the Pipeline library.
  2. To stop the pipeline, from the pipeline canvas, click the Stop icon.
    The Stop Pipeline Confirmation dialog box appears.
  3. To stop the pipeline, click Yes.
