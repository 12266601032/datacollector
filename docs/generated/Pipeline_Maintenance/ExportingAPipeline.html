Exporting a Pipeline

Export a pipeline when you want to use the pipeline with another Data Collector. You might also export a pipeline as a safeguard before you upgrade the Data Collector.

When you export a pipeline, you receive a JSON file named after the pipeline. For example, when you export a pipeline named MyPipeline, the exported file is named MyPipeline.json.

  1. In the Pipeline library, click the Actions icon for the pipeline that you want to export.
  2. Click Export.
    The Data Collector window generates the JSON file and saves it to your default downloads directory.
